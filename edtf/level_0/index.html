Time Interval

ISO 8601-1 specifies a “time interval” and this reflects that wording, but EDTF explicitly disallows time of day in intervals. So it’s more of a “date interval”.

The format is [date]/[date].

FormatExample
[date]/[date]1964/2008
2004-06/2006-08
2004-02-01/2005-02-08
2004-02-01/2005-02
etc
